How to Install, Uninstall, and Launch Apps on Flip Phones

This guide will teach you how to install apps on Android Flip Phones. You can see a list of apps we compiled on our apps page. We do not endorse any app or tell you any apps are okay. It is up for you to decide which apps are hashkafically right for you.


Need help? Join our forums, where our amazing community members can assist you with any questions!

Installing Apps

Step 1: Enable USB Debugging

Enabling USB debugging varies by device:

  • Kyocera & Non-LG Phones:
    1. Open Settings → Software Information
    2. Tap Build Number 7 times
    3. Return to Settings, find Developer Options, and enable USB Debugging
  • LG Exalt:
    1. Dial ##7764726220
    2. Enter 000000 as the service code
    3. Navigate to Developer Options → USB Debugging → Enable
  • LG Classic Flip & LG Wine 2:
    1. Dial ##228378 and press call
    2. Scroll to Developer Options
    3. Enable USB Debugging

Step 2: Install Apps

  • For Kyocera: Use Bluetooth to transfer the APK from another phone, then install from the received notification.
  • For LG Exalt: Download the APK from our site, transfer it to an SD card, and install via File Manager.
  • For Most Other Phones: Use ADB:
    1. Visit WebADB and click Connect.
    2. Use the Install APK feature to install the APK.
    3. See this guide on our forum for more information about using ADB.

Advanced Users: Install via ADB Command

For those using ADB on a computer: adb install -g appname.apk

Example: adb install -g waze.apk

See this guide on our forum for more information about using ADB.

Uninstalling Apps

  • LG Exalt:
    1. Dial ##7764726220
    2. Enter 000000 as the service code
    3. Navigate to Applications, select the app, and choose Uninstall
  • All Other Phones: Use ADB: adb uninstall package_name

    Example: adb uninstall com.android.browser

    Don't know the package name? Run the following command to find the package name. Replace app_name with one word from your app's name:

    pm list packages | grep app_name

    See this guide on our forum for more information about using ADB.

Launching Installed Apps

  • LG Exalt:
    1. Dial ##7764726220
    2. Enter 000000 as the service code
    3. Navigate to Applications to view and launch installed apps.
  • For All Other Phones: Install the Launcher APK from our site, then access installed apps via the launcher.
    Some phones allow you to replace apps on your home screen with other apps (like some Kyocera Models) which is another way off accessing them.
